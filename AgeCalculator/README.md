# Age Calculator – Zamzam

## 📖 Description
A modern and elegant web application to calculate age using **HTML, CSS, and JavaScript**.  
The interface features a full-screen video background with a centered content box, allowing users to enter their name, select their birthday, and calculate their age either manually or automatically.

---

## 🛠️ Technologies Used
- **HTML5**: Structure and layout.  
- **CSS3**: Styling, video background, content box design, and button effects.  
- **JavaScript (Vanilla)**: Input validation, age calculation logic, and interactive button states.

---

## 🚀 How to Run
1. Place all files in a project folder.  
2. Ensure the video file `background.mp4` is available in the correct path (or update the source in the HTML).  
3. Open `index.html` in your browser.  
4. Enter your name (letters only, Arabic or English).  
5. Select your birthday and choose either manual or automatic calculation.  

---

## ✨ Features
- Full-screen dynamic video background.  
- Centered transparent content box with shadow effects.  
- Stylish buttons with hover and disabled states.  
- Name validation (Arabic and English letters only).  
- Restriction against future dates to avoid negative age calculations.  
- Manual or automatic age calculation options.  
- Clear and interactive feedback messages.
