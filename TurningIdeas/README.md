# Turning Ideas into Apps – Zamzam

This project is a simple interactive web page built with **HTML, CSS, and JavaScript**.  
It introduces me, Zamzam Al Balushi, and allows visitors to interact through prompts.

## Features
- Responsive layout with a background image
- Interactive buttons:
  - **Share your name** → Greets the user with time-based messages (morning, afternoon, evening)
  - **How’s your day going?** → Collects user mood and responds with encouragement
  - **What do you specialize in?** → Displays user specialty with motivational text
- Styled with CSS for a clean and modern look
- JavaScript functions for dynamic interaction

## Technologies Used
- **HTML5** for structure
- **CSS3** for styling and layout
- **JavaScript** for interactivity
